BOM vs DOM in JavaScript

BOM vs DOM in JavaScript

Browser Object Model (BOM)

The Browser Object Model (BOM) represents the browser as a host environment. It provides objects and methods to interact with the browser window.

  • Examples include window, navigator, location, history, and screen.
  • BOM is not standardized and varies between browsers.
  • It allows manipulation of browser behavior, such as redirecting to a new page or adjusting browser settings.

Document Object Model (DOM)

The Document Object Model (DOM) represents the structured hierarchy of elements within a web page, allowing scripts to access and manipulate them.

  • It provides a structured representation of the HTML document, treating each element as an object.
  • DOM allows dynamic updating and interaction with the document content.
  • Examples include methods to select elements (getElementById, querySelector), modify content, and respond to events.
  • DOM is standardized by the W3C, ensuring consistent behavior across browsers.

How many type of Variable in JavaScript?

ANS.

We can declare variables to store data by using the var, let, or const keywords.

  1. let – is a modern variable declaration.
  2. var – is an old variable declaration method. Normally we don’t use it at all, but we’ll cover subtle differences from let in the chapter The old "var", just in case you need them.
  3. const – is like let, but the value of the variable can’t be changed.

syntex

var name = dhyanesh

let gender = male

const age = 25

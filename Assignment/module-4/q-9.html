Q-9 Which property controls the image scroll in the background?

Answer:-


↣ background-attachment property is used to controls the image fixed and scroll in the background.

↣ There are two properties of background-attachment there are fixed and scroll.

↣ By default value is set to scroll.

↣ background-attachment:- fixed

↣ background-attachment:- scroll


background-attachment:- fixed


background-attachment:- scroll

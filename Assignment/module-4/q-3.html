Q-3 What is the difference between CSS2 and CSS3?

Answer:-


CSS2 CSS3
CSS is the basic version with the basic formatting functionality. CSS3 is the latest version of CSS language that exceeds the functionality of CSS2.
It does not support responsive design and cannot handle media queries. It supports responsive design and can handle media queries as well.
CSS2 is relatively slower than CSS3 CSS3 is the latest verasion which is much faster than its pervious iterations.
It does not suppport 3D animations and transformations. You can create 3D transformations, transitions, and animations using CSS3.
It has old and standard colors. It offers exciting new ways to play with colors.

Q-7 How can image repetition of the backup be controlled?

Answer:-


Image repetition of the background can be controlled by using background-repeat property. By default the value is set to repeat.

  • repeat:-The background image is repeated both vertically and horizontally.
  • repeat-x:-The background image is repeated only horizontally.
  • repeat-y:-The background image is repeated only horizontally.
  • no-repeat:-The background-image is not repeated.

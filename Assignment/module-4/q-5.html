Q-5 What do you understand by CSS opacity?

Answer:-


  • ↣ The opacity property sets the opacity level for an element.
  • ↣ The opacity level describes the transparency level, where 1 is by default and 0 is completely transparent.
  • ↣ the opacity property to add transparency to the background of an element, all of its child element become transparent.

Q-17 What is the meaning of the CSS selector?

Answer:-


↣ A CSS selector means it will selects the HTML elements you want to style.

↣ There are differnet types of CSS selector there are as folows:-


Child selector

                    ul > li {
                        - Styles applied to all <li> elements that are direct children of a <ul>.
                        - It use > sign and space to select child element.
                        }


Class selector

                    .container {
                        - Styles applied to all class = "container" element.
                        - class selector is select by using .container. 
                        }


Id selector

                    #cointainer {
                        - Styles applied to all id="main" element.
                        - Id selector is select using #container.
                        }


Element selector

                    div {
                        - Styles applied to all the <div> elements.
                        - Element selector is select using element name.
                        }


Universal selector

                    * {
                        - Styles applied to all elements
                        - Universal selector is select by using *.
                        }


Group selector

                    p, div, h1 {
                        - Styles applied to all elements we have to style.
                        - Group selector is select by using ,.
                        - We can select multiple class, multiple id, multiple element.
                        }


General-sibling selector

                    h1 ~ p {
                        - Styles applied to <p> element that are sibling of <h1> element.
                        - general-sibling selector is selct by using ~.
                        }


Adjsent-sibling selector

                    h1 + p {
                        - Styles applied to directly after the specific element.
                        - Adjsent-sibling selector is selct by using +.
                        }


Pseudo selector:-

- Pseudo class

                    .container:hover {
                        - Styles applied to the class="container" element.
                        - The pseudo class is select using :.
                        - There are different type of pseudo class there are 
                        focus, active, visited, n-th child().
                        }

- Pseudo Element

                    .container::after {
                        - Styles applied to class="container" element.
                        - The pseudo element is select using ::.
                        - There are different type of  pseudo element there are 
                        after, before, first-letter, first-line, placeholder, selection
                        }

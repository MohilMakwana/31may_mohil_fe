Q-6 How can the background color of an element be changed?

Answer:-


The background color of the element be changed by using background-color property.

  • Name:-By using color name.
  • hex:-By using hexa value using # sign.
  • rgb:-By using rgb value full form is red, green and blue. value start from 0 to 255.
  • rgba:-By using rgba value full form is red, green, blue and alpha. alpha value start from 0 to 1
  • hsl:-By using hsl method stands for hue,saturation and lightness.
  • hsla:-By using hsla method stands for hue,saturation and lightness.
    • background-color: skyblue
    background-color: #a5cde3;
    background-color: rgb(226, 176, 176);
    background-color: rgb(226, 176, 176, 0.3);
    background-color: hsl(120, 50%, 50%);
    background-color: hsla(120, 50%, 80%, 0.7);

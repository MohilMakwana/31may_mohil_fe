Q-7 What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements? With Example.

Ans:-id attribute is unique we can not use same id to other value while class is used to Specifies one or more class names for an element.
- An ID is an attribute, a unique identifier assigned to only one HTML element within a page. It is often used for unique styling.
- The class attribute lets you give the same name to multiple HTML elements.
- Classes are not unique and can be assigned to multiple elements. They are generally used for applying the same styles or behaviors to a group of elements.

Eg.
id attribute Example.
ID attribute example 
        <div id="myUniqueID">This is a div with an ID.</div>
class attribute example 
        <div class="myClass">This is a div with a class.</div>
        <p class="myClass">This is a paragraph with the same class.</p>

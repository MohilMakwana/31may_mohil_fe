Q-14 What is the use of a span tag? Explain with example?

Ans:- Span tag is an inline element.
- Span tag is paired tag means it has both open tag and close tag.
- Span tag is easily styled by CSS using the class or id attribute.

Eg. 
        This is <span style="color: greenyellow;">Span Tag.</span>
    
        <span> Span tag is a inline element </span>

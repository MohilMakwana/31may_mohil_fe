Q-11 What is the difference between a block-level element and an inline element?

Ans:-
Inline element
- In inline element height and width is according to the content.
- Inline elements do not start on a new line and only take up as much width as necessary.
Eg. inline element - b, i , u, img, a etc.


Block level element
- While in the block level element width is full width of the screen.
- Block-level elements are those that start on a new line and take up the entire width of their container by default.
Eg. Block level element - p, br, blockquote, h1 to h6 etc.

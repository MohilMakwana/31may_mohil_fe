Q-5 What are different types of lists in HTML? With Example.

Ans:-There are three types of lists in HTML.
1.Ordered list - Used when you want to arrange the list into order formate.
2.Unordered List - Used when you don't want to arrange the list into order formate.
3.Dat List - used to display data and their definition.
Eg. Ordered List

  1. Ordered item 1
  2. Ordered item 2
  3. Ordered item 3

Unordered List
  • Unoredred item 1
  • Unordered item 2
  • Unordered item 3

Data list
Data List
Used to display data and their definition.

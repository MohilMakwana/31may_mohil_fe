Q-8 In Bootstrap 4, what is flexbox?

Answer:-


  • In Bootstrap 4, the flexbox is used to create flexible and responsive layouts.
    • Flex Direction
  • The .flex-row, .flex-row-reverse, .flex-column, and .flex-column-reverse classes are used to set the direction of the main axis and the cross axis for the flex container.

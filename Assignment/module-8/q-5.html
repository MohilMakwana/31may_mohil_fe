Q-5 What is the difference between Bootstrap 4 and Bootstrap 5

Answer:-


BASIS OF BOOTSTRAP 4 BOOTSTRAP 5
Grid System Bootstrap 4 has five levels: xs, sm, md, lg, and xl. Bootstrap 5 has six levels: xs, sm, md, lg, xl, and xxl..
Color Bootstrap 4 provides a few colors for a web page. Extra colors added with the looks
Icons Bootstrap 4 lacks SVG icons; we must rely on font-awesome for icons. Bootstrap 5 has its own set of SVG icons.
CSS usability Bootstrap 4 uses fewer CSS properties. Bootstrap 5 uses more CSS properties.
Vertical Groups Columns can be positioned relative to one another. Columns cannot be positioned relative to one another.

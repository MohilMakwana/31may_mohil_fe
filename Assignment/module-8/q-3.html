Q-3 What are the default Bootstrap text settings?

Answer:-


  • Bootstrap uses a default font-size of 1rem (16px by default), and its line-height is 1.5.
  • In Bootstrap text alignment set by it's class
  • The default font-family is "Helvetica Neue", Helvetica, Arial, sans-serif.
  • In Bootstrap fs-1 to fs-6 use for diffrent font size.
  • In Bootstrap text decoration set by it's class (".text-decoration-none",".text-decoration-underline".)
  • Bootstrap styles HTML headings (<h1> to <h6>).
  • Bootstrap styles HTML display (<d1> to <d6>).
  • In Bootstrap text decoration set by it's class (".text-decoration-none",".text-decoration-underline".)

Q-5 What are undeclared and undefined variables?

Answer:-


  • Undefined:-a variable has been declared but has not been assigned any value.

    • let x;

    output - undefined

    console.log(x)



  • Undeclared: when a variable is not declared earlier using the let,var or const keyword.

    • // ReferenceError: y is not defined

    console.log(y)

Q-2 How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

Answer:-


  • Applying CSS using Jquery
    •                 <script>
                    $(document).ready(function () {
                        $('div').css({
                            background: '#f4d23e',
                            color: '#f00'
                        });
                    });
                </script>

  • Add and Remove class in jquery
    •                     <script>
                        $(document).ready(function(){
                            $('.Add').click(function(){});
                            $('.Remove').click(function(){
                                $('p').remove();
                            });
                        });
                    </script>
  • Jquery Animation
    •                     <script>
                        jQuery(document).ready(function () {
                            jQuery('button').click(function () {
                                jQuery('div').animate({
                                    borderRadius: '50%',
                                    opacity: 'toggle',
                                    'height': '+=250',
                                    width: 'toggle'
                                }, 
                                {
                                    duration: 200,
                                    easing: 'linear',
                                    // complete: function () {
                                    //     alert("The animation is completed");
                                    // }
                                });
                                });
                        });
                    </script>

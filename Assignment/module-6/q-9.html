Q-9 How can the style/class of an element be changed?

Answer:-


  • We can change, add or remove any CSS property from an HTML element with the help of JavaScript.
  • style.property
    • syntax
    document.getElementById("id").style.property = value
    Eg.:- document.getElementById("id").style.color = #000000
  • Changing the class itself
    • syntax
    document.getElementById("id").classList
    we can use the methods to add classes, remove classes, and toggle between different classes respectively.

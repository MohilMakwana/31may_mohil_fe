Q-2 What is the use of isNaN function?

Answer:-


  • The meaning of 'isNaN' is "is Not A Number".
  • The JavaScript isNaN() Function is used to check whether a given value is number or not. It returns true if the value is a NaN else returns false.
  • If given values data type is 'Number' than this function give "false" value otherwise for other data types it return "true " value .

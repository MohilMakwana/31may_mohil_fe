Q-16 What are the disadvantages of using innerHTML in JavaScript?

Answer:-


  • Inner HTML is slow
  • Event handlers attached to any DOM element are preserved
  • When innerHTML property is used to modify, all the DOM nodes will have to be parsed and created again.
  • In JavaScript, ‘+=’ is commonly used for appending. However, when using innerHTML to append to an HTML tag, the entire tag is re-placed.
  • Used for Cross-site Scripting

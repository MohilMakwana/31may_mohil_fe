Q-11 How can you convert the string of any base to an integer in JavaScript?

Answer:-


  • In JavaScript parseInt() function (or a method) is used to convert the string parameter to an integer value itself.
  • parseInt() Method is used to accept the string parameter and convert it into an integer.
  • syntax - parseInt(Value)

  • the Number() method is used to convert any primitive data type to a number, if it is not convertible it returns NAN.
  • syntax - Number(value)

  • the Unary operator(+) is used to convert a string, boolean, and non-string to a number.
  • syntax - +string;

Q - 2 What are tags and attributes in HTML?

Ans:- Tags are used to tell the browser how to display the content.
          eg. <b> </b> , <i> </i> , <u> </u> etc.
          While the attributes is used to define the property of element or to add the additional information.
          eg. href attribute - <a href="https://www.amazon.com" >Amazon</a>
          src, width, and height attribute - <img src="img_girl.jpg" width="500" height="600"> etc.

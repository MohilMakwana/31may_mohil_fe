Q-10 How can we club two or more rows or columns into a single row or column in an HTML table? With Example.

Ans:- We can club two or more rows into single row by using 'rowspan' attribute and to club two or more columns into single column we use 'colspan'attribute.

Eg.
Use of Colspan
Weekend
Monday Tuesday Wednesday Thrusday Friday Saturday Sunday


Use of rowspan
Weekend Monday
Tuesday
Wednesday
Thrusday
Friday
Saturday
Sunday

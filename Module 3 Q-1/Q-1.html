Q - 1 Are the HTML Tags and Elements the same thing?

Ans:- No the HTML tags and elements are not the same things.
          Tags are defined as how browser display the content there are mainly two types of tags pair and unpair tags.
          eg. : <b> </b> , <i> </i> , <u> </u> etc.
          Elements are defined as the combination of starting tag, content and ending tag.
          eg. : <p> This is content </p> , <del> This is delete tag </del> etc.
